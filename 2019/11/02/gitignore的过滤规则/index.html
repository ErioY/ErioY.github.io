一、.gitignore文件的作用

如果希望一次性把所有的文件进行add。那么可以使用命名： git add .

但是，有些文件不需要放入版本库，如何解决，使用 .gitignore

在使用git的时候，项目目录下的.gitignore文件的作用是告诉git哪些文件不需要添加到版本管理中。

版本管理工具，管理源代码的版本，即开发目录下程序员写的代码，而不是发布目录下的

哪些不需要管理版本：.git文件夹， node_modules
