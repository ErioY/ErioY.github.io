js特殊函数和运算符

字数统计: 1.8k 字 | 阅读时长 ≈ 7 分钟

  • 自执行(立即执行/自调用)函数

  1. 自执行函数语法：

    这两种自执行函数的写法，在函数体后面加括号就能立即调用，这个函数必须是函数表达式，不能是函数声明

  2. 自执行函数可以正常传参，第一个括号是形参的位置，第二个括号是实参的位置；它的内部也是域，变量是局部变量，很多框架里使用自执行函数来防止全局变量的污染；我们也可以用自执行函数将循环for中的i锁定在自执行函数内，我们之前则是使用自定义属性来实现类似的效果的

  • 数学函数

  1. Math.floor()：向下取整(下舍去)

  2. Math.ceil()：向上取整(上进入)

  3. Math.round()：四舍五入

  4. Math.abs()：取绝对值

  5. Math.max()：比较(取大)

  6. Math.min()：比较(取小)

  7. Math.sin()： 正弦函数(参数为弧度)

  8. Math.cos()： 余弦函数(参数为弧度)

  9. Math.tan()： 正切函数(参数为弧度)

  10. Math.PI： 圆周率(无理数pi)

  11. Math.sqrt(x)： x的平方根(开方)

  12. Math.pow(x,y)： x的y次方(ES7中用x**y表示)

  13. Math.random()：0~1的一个随机数，下面是几个随机整数的公式：

  14. Num.toPrecision(x)： 将Num固定为x位数字(小数多余部分会四舍五入，整数多余部分会用科学计数法后四舍五入)

  15. Num.toFixed(x)： 将Num的小数部分取x位有效数字(多余部分会四舍五入，不足部分会补0)

  • 高阶函数和递归函数

  1. 函数其实都指向某个变量。函数的参数能接收变量，那么一个函数就可以接收另一个函数作为参数，这种函数就称之为高阶函数：

  2. 所谓的递归函数就是在函数体内调用本函数。使用递归函数一定要注意，处理不当就会进入死循环。递归函数只有在特定的情况下使用 ，比如阶乘问题：

  3. 但我们把函数指针改变并修改原指针时会出错，原因是递归调用的时候原指针已改变了。解决办法是在函数内部调用当前函数用arguments.callee代替函数名，callee属性的初始值就是正被执行的 Function 对象。它是 arguments 对象的一个成员，表示对函数对象本身的引用。

  4. 但是访问 arguments 是个很昂贵的操作，因为它是个很大的对象，每次递归调用时都需要重新创建。影响现代浏览器的性能，还会影响闭包。因此我们在使用递归的时候还是用函数名的写法，同时避免上面例子的写法就可以了

  • 运算符

  1. 算术：+(加)、-(减)、*(乘)、/(除)、%(求余)
  2. 三元运算：a ? b : c既if/else特殊写法，等同于if ( a ){ b }else( c )
  3. 赋值：=(将=右边的值赋给左边)、+=(将+=左右两边求和后赋给左边)、-=、*=、/=、%=(后面的如同+=)
  4. 判断：<、>、<=、>=、==、!=、===、!==，=== 和 !== 只有在相同类型下,才会比较其值，类型不同直接返回布尔值
  5. 逻辑：&&——逻辑”与”操作；||——逻辑”或”操作；!——取反操作
    • a&&b逻辑”与”定义：从左向右读到false就返回(后面数据不会继续判断)，如所有数据都是ture则返回最后一个(因此都为true时返回的是true)
    • a||b逻辑”或”定义：从左向右读到true就返回(后面数据不会继续判断)，如所有数据都是false则返回最后一个(因此都为false返回的是false)
    • 注意：虽然返回结果是根据布尔值决定的，但js的逻辑运算返回的是数据，而不是布尔值，这与其他编程语言不一样(如PHP返回的就永远是布尔值)
  6. 运算优先级：()可以让其内部表达式先执行，如算式或逻辑运算等
