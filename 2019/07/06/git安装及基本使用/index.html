github里，找到settings，填入秘钥

正式开始

1、确定git和github已经连接

2、本地文件推到github上

  • 将新建的文件添加到git的缓存区
  • 给提交的文件添加描述
  • 汇总到总分支上
  • 退出
  • 文件推到远程仓库
  • 输入用户名(如果设置了SSH密匙的话就不用输入密码了)
