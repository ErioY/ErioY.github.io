  • 事件对象event

  1. 当我们与浏览器中 Web 页面进行某些类型的交互时，事件就发生了。可能是用户在某些内容上的点击、鼠标经过某个特定元素或按下键盘上的某些按键。还可能是 Web 浏览器中发生的事情，比如某个 Web 页面加载完成，或者是用户滚动窗口或改变窗口大小。

  2. 当事件发生时，和这个对象事件有关的一些信息都会被临时保存到一个指定地方–event对象(类似飞机的黑匣子)

  3. 注意：事件对象必须在一个事件调用的函数里面使用才有内容。函数是不是事件函数，不是定义时决定，而是调用时决定，如：
