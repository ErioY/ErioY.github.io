swan-core 插桩

下载好的 box.zip 就是插了桩的 swan-core ，替换手百 swan-core 后，执行统计行覆盖率的命令：npm run cts -- --cov，之后会生成一个json文件，包含 swan-core 在本地的路径和行覆盖，而只有在本机才可以看到具体文件（比如缺少了哪些具体的用例），其他电脑只能查看覆盖率，看不到具体文件

  1. 在 cos-swan 文件根目录执行命令：sh buildSwanCore.sh temp(无用)

    npm run build-swan

http://cp01-swn.epc.baidu.com:8090/static/zip/2019110510485451/box.zip

![image-20200116101602925](/Users/wangwenhan/Library/Application Support/typora-user-images/image-20200116101602925.png)

  1. 将上图中的地址生成二维码
  2. 在百度包管理平台（http://package.baidu-int.com:8080/#/main?type=1）中找到新版本，打开手百，扫描二维码，下载并安装，打开开源宿主调试工具，找到最下面的“替换swan-core”，扫描第2步中的二维码并替换
  3. 打开 cts-swan 文件，将 buildSwanCore.sh 文件里的路径修改为 cts-swan 文件所在的路径，保存并退出 （下载在本机的包不需要这一步）
